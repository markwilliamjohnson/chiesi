The notebook provides helper functions to safely load and examine data from different sources:

# Helper function to safely read CSV files
def safe_read_csv(path: str, **kwargs) -> pd.DataFrame:
    """Read a CSV with friendly errors and sensible defaults."""
    if not os.path.exists(path):
        raise FileNotFoundError(f"CSV not found: {path}")
    defaults = {
        "encoding": "utf-8",
        "low_memory": False,
    }
    defaults.update(kwargs)
    return pd.read_csv(path, **defaults)
Key Learning Point: Good data code includes error handling and sensible defaults. The safe_read_csv function checks if files exist before trying to read them, preventing crashes.

Filtering and Selecting Target Data

With many drugs and conditions, we need to focus on a specific area. The notebook shows how to filter data:

# Example filtering template for target drug class
TARGET_KEYWORD = "oncology"
TARGET_CODE_PREFIX = None

if drug_catalog is not None:
    df = drug_catalog.copy()
    cols_text = [c for c in df.columns if df[c].dtype == object]

    mask_kw = pd.Series(False, index=df.index)
    if TARGET_KEYWORD and cols_text:
        for c in cols_text:
            mask_kw = mask_kw | df[c].astype(str).str.lower().str.contains(TARGET_KEYWORD.lower())

    target_drugs = df[mask_kw]

Building a Market Scoring Model

The core of the hackathon is creating a market score to prioritize countries. The notebook shows a simple formula:

# Simple market scoring formula
def normalise_series(s: pd.Series) -> pd.Series:
    s = pd.to_numeric(s, errors="coerce")
    if s.notna().sum() == 0:
        return s
    return (s - s.min()) / (s.max() - s.min() + 1e-9)

# MarketScore = w1·(Burden) + w2·(Affordability) + w3·(Access) − w4·(Risk)
w_burden, w_aff, w_access, w_risk = 0.45, 0.35, 0.00, 0.20
market["MarketScore"] = (w_burden * burden) + (w_aff * aff) - (w_risk * risk)
Key Learning Point: Data normalization (scaling values between 0-1) allows us to combine different metrics (like prevalence and GDP) that have different units and scales.

# Example: Finance scenario calculator
def market_sizing_scenarios(
    patients: pd.Series,
    annual_price: float,
    penetration_rates: Dict[str, float],
) -> pd.DataFrame:
    rows = []
    for scen in penetration_rates.keys():
        pen = penetration_rates[scen]
        revenue = patients * pen * annual_price
        rows.append({"scenario": scen, "revenue": revenue})
    return pd.DataFrame(rows)
